## 介绍

统计某目录下的所有文本文件的单词出现频率。支持大驼峰和小驼峰命名的单词组合拆分，例如：```void setName(name);```或者void SetName(Name);```将被分解成4个单词。只支持英文单词，一个字母的单词将忽略。只打印前100的数据，可自行定义。
